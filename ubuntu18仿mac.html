美化步骤

1.gnome tweak tool 安装^1

sudo apt-get install gnome-tweak-tool

2.gnome shell 安装^2

安装使用主题必备的扩展user themes ，地址

其他扩展推荐：

3.主题下载

使用的是mac osc，地址：https://www.gnome-look.org/p/1171688/ 这里我打包好了，包括壁纸、图标、shell，下载地址

4.安装主题

我的是在当前用户下创建的主题，所以只对当前用户有效，在/home/{你的名字} 下面创建.icons 、.themes 文件夹 .icons放置的是图标，.themes里面放的是主题和Shell。 我打包的可直接在目录下解压即可，然后打开tweak tool（名字显示叫调试），在外观下选择下图对应的即可

5.标题栏按钮放左边

点击 “窗口”——“标题栏按钮”——“左”
